\section*{How to Compile}

All figures use TikZ. Compile on Overleaf or locally:

\begin{verbatim}
# Pipeline diagram (Fig. 1)
pdflatex pipeline_tikz.tex

# Domain adaptation effect (Fig. 2)
pdflatex fig_domain_adapt.tex

# Ablation + comparison tables (Table 1, 2)
pdflatex table_ablation.tex
\end{verbatim}

On Overleaf, select XeLaTeX as compiler.
